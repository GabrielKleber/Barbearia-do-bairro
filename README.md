<h1>💈 Landing Page – Barbearia do Bairro</h1>

Este projeto é uma landing page fictícia criada como modelo de apresentação para barbearias locais que desejam ter presença profissional na internet.

A página foi desenvolvida com foco em conversão de clientes, destacando serviços, depoimentos, galeria de trabalhos e botão direto para contato via WhatsApp.

<h2>🎯 Objetivo do projeto</h2>

Demonstrar, de forma prática, como uma landing page simples e bem estruturada pode ajudar uma barbearia a:

Atrair novos clientes
Transmitir profissionalismo
Mostrar serviços e resultados
Facilitar o agendamento rápido pelo WhatsApp
<h2>🧩 Seções da página</h2>

A landing page contém:

Hero section com chamada principal e botão de agendamento
Quem somos com história da barbearia e imagem dos fundadores
Serviços com cards modernos inspirados em sites reais do nicho
Galeria de trabalhos (cortes, barba, sobrancelha, etc.)
Depoimentos de clientes
Horários de funcionamento
CTA final com botão de WhatsApp
🛠️ Tecnologias utilizadas
HTML5
CSS3 (Flexbox + Grid)
JavaScript puro (Vanilla JS)
<h2>💡 Observação</h2>

Este projeto é apenas demonstrativo e pode ser facilmente adaptado para clientes reais, alterando textos, imagens e informações de contato.
